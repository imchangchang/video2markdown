## P0 - 严重问题

### API 超时导致长视频处理失败

**问题描述**: 两个长视频（20分钟、14分钟）在 Stage 6 失败，错误为 API 超时

**错误日志**:
```
httpx.ReadTimeout: The read operation timed out
openai.APITimeoutError: Request timed out.
```

**根本原因**: 
- Stage 6 需要将所有文字稿、图片分析结果一次性发送给 AI
- 长视频内容量大，AI 处理需 3-6 分钟
- 原超时 120s 不足

**解决方案**: 超时 120s → 900s
