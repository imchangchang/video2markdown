# 05-配置命名更新

## 变更内容

将配置文件中的 `KIMI_` 前缀统一改为 `VIDEO2MD_`

## 配置对照表

| 旧名称 | 新名称 |
|--------|--------|
| `KIMI_API_KEY` | `VIDEO2MD_API_KEY` |
| `KIMI_MODEL` | `VIDEO2MD_MODEL` |
| `KIMI_VISION_MODEL` | `VIDEO2MD_VISION_MODEL` |
| `KIMI_BASE_URL` | `VIDEO2MD_BASE_URL` |
| `KIMI_ASR_PROVIDER` | `VIDEO2MD_ASR_PROVIDER` |
| `KIMI_WHISPER_MODEL` | `VIDEO2MD_WHISPER_MODEL` |

## 向后兼容

`config.py` 中已实现向后兼容：
```python
def model_post_init(self, __context):
    if not self.api_key:
        if old_key := os.environ.get("KIMI_API_KEY"):
            self.api_key = old_key
```
